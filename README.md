Vertical Slider

Preview ![Preview](preview.png)

It's an interesting slider in which you can drag slides in "y" coordinate. You also can see all the images in the left side of the website and if you click on them you'll see the slide in which they are. And those small windows with those mini-photoes have an animation.

I used HTML, CSS and JS for this slider. The slides are in a BIG div, and the items of that div are located in column. So when you drag the slide in y coordinate you see tha following or previous slide.
